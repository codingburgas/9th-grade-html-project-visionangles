This page presents the main features of our application, created as a project for Information Technology. The goal is to develop a functional system for reporting fires and coordinating teams.

Automatic Location Detection

When a report is submitted, the system automatically detects the user's location via GPS or IP geolocation. This is crucial for a quick response from teams, eliminating the need to clarify the address and saving valuable time.

Adding Photos to Reports

Users can attach photos of the fire or the affected area. This allows responders to assess the scale and specifics of the situation before arriving on-site. The feature is implemented using an HTML input of type 'file' and JavaScript for visualization.

Sending Notifications
